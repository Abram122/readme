1. Fundamentals of HTML and XHTML HTML is the standard markup language for creating web pages.
It consists of a series of elements.
HTML is web design language and not programming language.
Hypertext Markup Language (HTML) documents use the .html or .htm extension.
This extension allows a web browser or device such as a smartphone to understand
that HTML content is on the page, and the content of the page is then rendered
by the browser or deviceaccording to the rules of HTML.
There is little fundamental difference between HTML and XHTML
(EXtensible HyperText Markup Language)
the two standards previously released by the W3C (World Wide Web Consortium).
As XHTML was defined, it was created so that pages written in XHTML
also work in browsers that render current HTML.
The tags and attributes of XHTML and HTML remained the same,
but thesyntax of XHTML code is more strict.
The most significant differences between XHTML and HTML are as follows:
In XHTML, all tags must be lowercase.
XHTML requires all tags to be closed
meaning that there must be a tag at the startand end of the element being tagged
